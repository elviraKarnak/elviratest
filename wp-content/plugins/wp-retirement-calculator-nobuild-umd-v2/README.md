# WP Retirement Calculator (No-Build UMD v2)
- Adds **chart toggle**: "Super balance" vs "Income".
- No Node/npm required.
- Shortcode: `[retirement_calculator]`

## Install
1. Upload the plugin folder to `wp-content/plugins/` or use the ZIP.
2. Activate **WP Retirement Calculator (No-Build UMD v2)**.
3. Add `[retirement_calculator]` to a page.
