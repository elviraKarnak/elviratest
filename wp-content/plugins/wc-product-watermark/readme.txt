== Description ==

WC Product Watermark adds a lightweight, configurable watermark overlay to WooCommerce product images to protect your photos and promote your brand.

Features:
* Simple admin UI under **WooCommerce → Product Watermark**
* Choose watermark image using the Media Library
* Control position, opacity, and size (percentage)
* Lightweight frontend output with no heavy image processing
* Translation-ready

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wc-product-watermark` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to **WooCommerce → Product Watermark** and configure the watermark image and options.
