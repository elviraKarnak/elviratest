== Description ==

Elvira Quick Chat Button is a lightweight and easy-to-use plugin that adds a WhatsApp chat button to your WordPress website. Let your visitors contact you directly via WhatsApp with just one click!

= Free Features =

* Floating WhatsApp button (customizable position)
* Custom pre-filled message
* Mobile-only display option
* Customizable button text
* 4 position options (corners)
* Shortcode support: [whatsapp_chat]
* Translation ready

= Pro Features =

* Multiple agents with round-robin distribution
* Custom icons & advanced styling
* Advanced analytics dashboard
* Conditional display rules

= Shortcode Usage =

Basic usage:
`[whatsapp_chat]`

With custom parameters:
`[whatsapp_chat phone="919999999999" message="Hello!" text="Chat Now" style="button"]`

Available styles: button, link, icon

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/elvira-quick-chat-button` directory, or install through WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to WhatsApp Chat in your WordPress admin menu
4. Configure your WhatsApp number and settings
5. Save changes and check your website!

For Pro features:
1. Purchase a license from https://elvirainfotech.com
2. Go to Settings > EWQC License
3. Enter your license key and click Activate
4. Enjoy premium features!

== Frequently Asked Questions ==

= How do I add my WhatsApp number? =

Go to WhatsApp Chat in your WordPress admin menu and enter your phone number with country code (e.g., 919999999999 for India).

= Can I use a shortcode instead of the floating button? =

Yes! Use `[whatsapp_chat]` anywhere in your content. You can also disable the floating button in settings.

= Does this work on mobile devices? =

Yes! The plugin automatically detects mobile devices and opens the WhatsApp app. On desktop, it opens WhatsApp Web.

= How do I get Pro features? =

Visit https://elvirainfotech.com to purchase a license, then activate it in Settings > EWQC License.

= Is the plugin GDPR compliant? =

Yes, the free version doesn't collect any user data. Pro version analytics can be disabled if needed.
