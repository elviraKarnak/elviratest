Sticky notification bar for WordPress — lightweight, jQuery-free, show announcements, deals, or free-shipping offers with dismiss and page targeting.

== Description ==
Elvira Smart Notification Bar adds a small, fast, and accessible sticky announcement bar to your site. It’s built to be lightweight and easy to configure from the WordPress admin.

Free features:
* Single global bar
* Color pickers
* Dismiss with memory (per user)
* Position: Top / Bottom
* Page targeting: All / Home / Cart / Checkout

Pro features (unlock with a valid license key):
* Multiple scheduled bars (Custom Post Type)
* Start/end scheduling per bar
* Countdown timer per bar
* WooCommerce sale-aware bars

== Installation ==
1. Upload the plugin folder to `wp-content/plugins/`
2. Activate the plugin through the **Plugins** screen in WordPress
3. Go to **Settings → Elvira Smart Notification Bar** to configure
4. To enable Pro features, go to **Settings → ESNB License** and enter your license key

== Frequently Asked Questions ==
= How do I enable Pro features? =
Install the Pro upgrade and enter your license key at **Settings → ESNB License**. The UI will show Pro-only controls once a valid license is detected.


= Will this work with my theme? =
Yes — the bar is lightweight and uses unobtrusive CSS. If your theme has very unusual header positioning you may need a small CSS tweak, contact support if needed.
